# Mini_grocery_delivery_app
Developed a full-stack mini grocery delivery application inspired by Blinkit, enabling users to browse products, manage their shopping cart, and place orders through a responsive and user-friendly interface. Implemented secure user authentication, product management, order processing, and real-time cart updates. Designed the application with a clean architecture to ensure scalability, maintainability, and a seamless user experience.

Key Features:

User Registration and Secure Login
Product Browsing with Categories
Search and Filter Functionality
Shopping Cart Management
Order Placement and Checkout
Responsive UI for Desktop and Mobile
Clean and Modular Code Architecture

Tech Stack:

Frontend: HTML, CSS, JavaScript, React.js
Backend: Java, Spring Boot
Database: MySQL
Version Control: Git & GitHub
